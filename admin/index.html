Panel de Administración

Ingresa tu token de GitHub para gestionar las noticias.

¿Cómo obtengo el token?

1. Ve a github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Haz clic en Generate new token
3. Ponle un nombre (ej: Admin Cupey)
4. En Resource owner selecciona ocancel74
5. En Repository access elige Only select repositories y selecciona hogardeninasdecupey-wp
6. En Permissions busca Contents y cámbialo a Read and write
7. Haz clic en Generate token, copia el código y pégalo aquí

Solo necesitas hacer esto una vez. El token se guarda automáticamente.
